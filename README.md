# project_boilerplate

```
/project_template
	/app
	    /javascript
	    /less
	    /public
		    index.html
		    css.css
		    js.js
```
### Install
```sh
$ npm install
```

### Gulp

##### Watch file change with live reload and serve
```sh
$ gulp
```

##### Build the production js and cs
```sh
$ gulp build
```

##### Alternative start serve with node

```sh
$ node app.js
```
